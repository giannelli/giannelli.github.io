Proposition5.2.2

\(A'\) is a subspace if and only if \(\vec{b} \in A\).

Proof

If \(A \subset \mathbb{R}3\) be a subspace of rank 2 then it is a plane through the origin. If \(\vec{x} \in A\) then \begin{equation*}\vec{x} = t_1 \vec{v_1} + t_2 \vec{v_2}\end{equation*} for direction vectors \(\vec{v_1},\vec{v_2}\).

If \(\vec{b} \in A\)and \(\vec{b} \ne 0\) then \begin{equation*}\vec{b} = b_1 \vec{v_1} + b_2 \vec{v_2}\end{equation*} and so \(A'\) is \begin{equation*}A′ = \{ t_1\vec{v_1} + t_2 \vec{v_2} + b_1 \vec{v_1} + b_2 \vec{v_2} | t_1,t_2 \in \mathbb{R} \}\end{equation*} \begin{equation*} = \{ (t_1 + b_1)\vec{v_1} + (t_2 + b_2)\vec{v_2} | t_1,t_2 \in \mathbb{R} \}\end{equation*} \begin{equation*} = \{ s_1 \vec{v_1} + s_2 \vec{v_2} | s_1,s_2 \in \mathbb{R} \}\end{equation*} \begin{equation*} = A\end{equation*} Hence \(A'\) is also a plane through the origin.

If \(\vec{b} \notin A\) then \begin{equation*}A′ = \{ \vec{b} + t_1 \vec{v_1} + t_2 \vec{v_2} | t_1,t_2 \in \mathbb{R} \}\end{equation*} which is a plane not through the origin, and so is not a subspace.
