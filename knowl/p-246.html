If b ∈ A then it is a linear combination of vectors already in A and for all a ∈ A it follows that a+b∈A. Hence A′ ⊆ A. Let a ∈A then since b ∈ A it follows that a−b ∈ A. Therefore (a−b)+b ∈ A′. Hence A ⊆ A′. Thus A′ = A. Since A is a subspace, it follows that A′ is a subspace. If b ̸ ∈A and b ̸= 0 then for no value of a ∈ A is −a=b. In particular a+b ̸= 0. This shows that 0 ̸= A′ and so A′ cannot be a subspace.
